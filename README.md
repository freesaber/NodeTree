# NodeTree
使用nodejs+mongodb开发的一个微信公众号，买家可以发布求购信息，卖家可以发布出售信息。

参考项目：[https://github.com/freesaber/NodeLearning/tree/master/scott-film](https://github.com/freesaber/NodeLearning/tree/master/scott-film)
